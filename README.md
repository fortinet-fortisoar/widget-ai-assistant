## Overview

The FortiSOAR AI Assistant Widget adds the 'Advisor' bot to both the list view and detail view of modules in FortiSOAR, as well as in the playbook designer. The 'Advisor' bot is a powerful tool designed to provide contextual answers to your questions regarding security threats, response processes, work plans, Jinja expressions, and more! Additionally, it improves user experience and expedites the playbook creation and management process within the FortiSOAR platform.
